Core Concepts

  • Chemical equation: symbolic representation with formulas and coefficients.
  • Law of Conservation of Mass: atoms not created/destroyed, just rearranged.
  • Balancing: use coefficients only, never change subscripts.
  • Types: combustion, decomposition, combination, displacement, double displacement, redox.

Formulas & Equations

  • 4Na + O₂ → 2Na₂O
  • C + O₂ → CO₂
  • 2H₂ + O₂ → 2H₂O

Common Mistakes to Avoid

  • Changing subscripts instead of coefficients
  • Forgetting to balance
  • Confusion between exothermic and endothermic

Worked Examples

Example 1: Balance: Fe + O₂ → Fe₂O₃

Expected answer: 4Fe + 3O₂ → 2Fe₂O₃

Solution: Left: 4 Fe, 6 O. Right: 4 Fe, 6 O. Balanced!

Example 2: Type of CuO + H₂ → Cu + H₂O?

Expected answer: Displacement/Reduction

Solution: H reduces CuO, one element replaces another.

Practice Questions

EASY Q1: Balance: H₂ + Cl₂ → HCl
Answer: H₂ + Cl₂ → 2HCl
EASY Q2: Combustion of methane: CH₄ + ? → CO₂ + H₂O
Answer: CH₄ + 2O₂ → CO₂ + 2H₂O
EASY Q3: Heat released means?
Answer: Exothermic
MED Q4: Balance: P₄ + O₂ → P₄O₁₀
Answer: P₄ + 5O₂ → P₄O₁₀
MED Q5: Type: Zn + CuSO₄ → ZnSO₄ + Cu?
Answer: Displacement
MED Q6: What conserves in reactions?
Answer: Mass (atoms)
HARD Q7: Balance redox: C + O₂ → CO₂
Answer: C + O₂ → CO₂
HARD Q8: Balance: C₂H₆ + O₂ → CO₂ + H₂O
Answer: 2C₂H₆ + 7O₂ → 4CO₂ + 6H₂O
HARD Q9: Identify type: CaCO₃ → CaO + CO₂
Answer: Decomposition
