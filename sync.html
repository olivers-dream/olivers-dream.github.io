How to move progress from the old computer

  1. Open this same `Sync` page on the child’s current computer.
  2. Create or sign in to the same cloud account on both computers.
  3. On the old computer, click `Push This Device To Cloud`.
  4. On your computer, sign in with the same account. The latest progress will pull down automatically.
  5. If both computers already have different progress, export a backup JSON from each one before pushing.
  6. Then import the older backup here and merge it into the cloud record.

Firebase Setup

Cloud sync is ready in the portal, but it needs your Firebase project details once. Edit `Study_Portal/assets/cloud-config.js` and paste your Firebase web config there.

Enable `Email/Password` in Firebase Authentication and add the Firestore rules from `Study_Portal/FIREBASE_SETUP.md`.
