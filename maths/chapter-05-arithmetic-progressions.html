Book Chapter Summary (Easy Language)

You learn sequence patterns with constant difference, nth term, and sum formulas. This chapter appears in direct and word-problem forms in board exams.

  • Identify arithmetic progression from constant difference patterns.
  • Use nth-term formula for direct term-position questions.
  • Use sum formula to compute large totals without manual addition.
  • Model day-to-day contexts (savings, seating, pattern growth) using AP.
💡 Remember It Like This: Like climbing stairs of equal step height. Each new term goes up by same amount.

Key Points You Must Know

  • AP has fixed common difference d.
  • nth term gives position value quickly.
  • Sum formula saves repetitive addition.
  • Word problems often convert to AP form.

Formula / Theorem Quick Box

  • a_n = a + (n−1)d
  • S_n = n/2[2a + (n−1)d]

Note to Remember

  • Use (n-1)d, not nd.
  • d must be constant.
  • Use Sn formula for long sums.

Common Mistakes to Avoid

  • Using n instead of n−1 in nth term.
  • Mixing up nth term and sum formulas.

Worked Examples (Attempt First, Then Reveal)

Example 1: Find 20th term of AP 3, 7, 11, ...

Expected final answer format: 79

Step-by-step solution: a=3, d=4. a20 = 3 + 19×4 = 79.

Example 2: Find S15 for AP 2,5,8,...

Expected final answer format: 345

Step-by-step solution: S15 = 15/2[2×2 + 14×3] = 15/2[46] = 345.
