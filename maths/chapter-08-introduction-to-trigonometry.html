Book Chapter Summary (Easy Language)

Defines trigonometric ratios in right triangles and proves identities. Foundation for heights-distances and advanced math.

  • Define all six trigonometric ratios in right triangles.
  • Memorize and apply fundamental identities in simplification.
  • Use complementary angle relations for quick conversion.
  • Practice standard angle values for fast non-calculator solving.

Key Points You Must Know

  • sin, cos, tan are angle-based side ratios.
  • Ratios stay same for same angle, any triangle size.
  • Use identities to simplify expressions.
  • Complementary angle relationships are important.

Formula / Theorem Quick Box

  • sin²θ + cos²θ = 1
  • 1 + tan²θ = sec²θ
  • 1 + cot²θ = cosec²θ

Note to Remember

  • SOH-CAH-TOA recall helps speed.
  • sin²θ+cos²θ=1 is core identity.
  • Complementary angles swap co-functions.

Common Mistakes to Avoid

  • Mixing opposite/adjacent sides.
  • Wrong reciprocal pairs (sec↔cos, cosec↔sin, cot↔tan).

Worked Examples (Attempt First, Then Reveal)

Example 1: If opposite=3, hypotenuse=5, find sinθ.

Expected final answer format: 3/5

Step-by-step solution: sinθ = opposite/hypotenuse = 3/5.

Example 2: If sinθ=12/13, find cosθ.

Expected final answer format: 5/13

Step-by-step solution: From 5-12-13 triangle, adjacent/hypotenuse = 5/13.

Solution: 5/4

Solution: 0

Solution: 1/√2
HARD Q7: Prove 1+tan²θ=sec²θ.
Solution: Divide sin²θ+cos²θ=1 by cos²θ.
