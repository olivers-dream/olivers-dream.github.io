Book Chapter Summary (Easy Language)

This chapter teaches polynomial structure, degree, zeros, graph meaning, and relation between roots and coefficients for quadratics. It helps in equation-solving and function thinking.

  • Build polynomial basics: term, coefficient, degree, and standard forms.
  • Interpret zeros algebraically (p(x)=0) and graphically (x-axis intersections).
  • Use root-coefficient relations for quadratics to avoid lengthy factorization.
  • Apply division algorithm and factor theorem in proof and objective questions.
💡 Remember It Like This: Polynomial is like a machine: input x goes in, output p(x) comes out. Zeros are input values where machine output becomes zero.

Key Points You Must Know

  • A polynomial has non-negative integer powers only.
  • Zero means p(x)=0, graph touches/crosses x-axis there.
  • For ax²+bx+c, if roots are α,β then α+β=−b/a and αβ=c/a.
  • Division algorithm: p(x)=g(x)q(x)+r(x).

Formula / Theorem Quick Box

  • α + β = -b/a
  • αβ = c/a

Note to Remember

  • Zero means p(x)=0 exactly.
  • Sign in α+β is negative.
  • Check factors using substitution.

Worked Examples (Attempt First, Then Reveal)

Example 1: Find zeros of x² - 5x + 6.

Expected final answer format: 2,3

Step-by-step solution: Factor: x² - 5x + 6 = (x-2)(x-3). So zeros are 2 and 3.

Example 2: For 2x² + 7x + 3, find α+β.

Expected final answer format: -7/2

Step-by-step solution: For ax²+bx+c, α+β = -b/a = -7/2.

Practice with Solutions

EASY Q1: Find zeros of x² - 9.
Solution: x = ±3.
EASY Q2: Evaluate p(1) for p(x)=x²+2x.
Solution: 3
EASY Q3: Degree of polynomial 5x³+2x+1?
Solution: 3
MED Q4: If zeros are 4 and -2, form polynomial.
Solution: x² -2x -8
MED Q5: Factor x² - 7x + 10.
Solution: (x-2)(x-5)
MED Q6: Find relationship between roots and coefficients for 3x²-9x+6.
Solution: Sum=-3, product=2
HARD Q7: Divide 2x³+3x²-11x-6 by (x-2).
Solution: Quotient 2x²+7x+3, remainder 0.
HARD Q8: For x²+px+q with roots α and 2α, find p in terms of α.
Solution: p = -3α
HARD Q9: If p(x)=x⁴-1, find all zeros.
Solution: ±1
