Key Points You Must Know

  • Mean shows balance point of data.
  • Median is middle value position-based.
  • Mode is most frequent class/value.
  • Ogive helps estimate median graphically.

Formula / Theorem Quick Box

  • Median = l + [(n/2−cf)/f]h
  • Mean (step deviation): x̄ = a + (Σf_i u_i/Σf_i)h

Note to Remember

  • Median class from cumulative frequency.
  • Class mark needed for mean.
  • Mode means highest frequency.

Worked Examples (Attempt First, Then Reveal)

Example 1: Mean of 2,4,6,8,10?

Expected final answer format: 6

Step-by-step solution: Mean = (2+4+6+8+10)/5 = 30/5 = 6.

Example 2: Mode of 3,4,4,5,6?

Expected final answer format: 4

Step-by-step solution: Most frequent value is 4, so mode = 4.

EASY Q1: Median of 5,7,9?
Solution: 7
EASY Q2: Mean of 10,20,30?
Solution: 20
EASY Q3: Mode of 5,5,6,7,7,7,8?
Solution: 7
MED Q4: Write grouped-data median formula.
Solution: l + [(n/2−cf)/f]h
MED Q5: For grouped data, what is class mark?
Solution: (lower limit + upper limit)/2
MED Q6: Cumulative frequency purpose in data analysis?
Solution: Find median class and create ogive curve.
HARD Q7: When to use step deviation mean?
Solution: When class marks are large, for faster calculation.
HARD Q8: Construct frequency polygon from histogram.
Solution: Join midpoints of top sides of histogram rectangles.
HARD Q9: For 5 values, median position. For 6 values, median position.
Solution: Position = (5+1)/2=3rd | Position = average of 3rd and 4th values
