You summarize data using mean, median, mode and understand grouped distributions with cumulative frequency and ogive curves.

Section A - NCERT Style Practice

Solve in notebook first. Use reveal only to verify method.

A1. Mean of 2,4,6,8,10?

Expected final answer: 6

Steps: Mean = (2+4+6+8+10)/5 = 30/5 = 6.

A2. Mode of 3,4,4,5,6?

Expected final answer: 4

Steps: Most frequent value is 4, so mode = 4.

Section B - Mixed Difficulty Drill

EASY B1. Median of 5,7,9?

Answer: 7

EASY B2. Mean of 10,20,30?

Answer: 20

EASY B3. Mode of 5,5,6,7,7,7,8?

Answer: 7

MED B4. Write grouped-data median formula.

Answer: l + [(n/2−cf)/f]h

MED B5. For grouped data, what is class mark?

Answer: (lower limit + upper limit)/2

MED B6. Cumulative frequency purpose in data analysis?

Answer: Find median class and create ogive curve.

HARD B7. When to use step deviation mean?

Answer: When class marks are large, for faster calculation.

HARD B8. Construct frequency polygon from histogram.

Answer: Join midpoints of top sides of histogram rectangles.

HARD B9. For 5 values, median position. For 6 values, median position.

Answer: Position = (5+1)/2=3rd | Position = average of 3rd and 4th values

Quick Formula + Error Revision

  • Median = l + [(n/2−cf)/f]h
  • Mean (step deviation): x̄ = a + (Σf_i u_i/Σf_i)h

Common mistakes to avoid

  • Choosing wrong median class from cumulative frequency.
  • Ignoring class marks in grouped mean.
