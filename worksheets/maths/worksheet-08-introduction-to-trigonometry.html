Chapter Context (Detailed Breakdown)

Defines trigonometric ratios in right triangles and proves identities. Foundation for heights-distances and advanced math.

  • Define all six trigonometric ratios in right triangles.
  • Memorize and apply fundamental identities in simplification.
  • Use complementary angle relations for quick conversion.
  • Practice standard angle values for fast non-calculator solving.

Remember Before Solving

  • SOH-CAH-TOA recall helps speed.
  • sin²θ+cos²θ=1 is core identity.
  • Complementary angles swap co-functions.

Boxed Items (Must-Memorize)

  • Six Trig Ratios
  • Core Trig Identities
  • Complementary Relations

Section A - NCERT Style Practice

Solve in notebook first. Use reveal only to verify method.

A1. If opposite=3, hypotenuse=5, find sinθ.

Expected final answer: 3/5

Steps: sinθ = opposite/hypotenuse = 3/5.

A2. If sinθ=12/13, find cosθ.

Expected final answer: 5/13

Steps: From 5-12-13 triangle, adjacent/hypotenuse = 5/13.

Section B - Mixed Difficulty Drill

EASY B1. tanθ in terms of sin and cos?

Answer: sinθ/cosθ

EASY B2. sin30°=?

Answer: 1/2

EASY B3. cos60°=?

Answer: 1/2

MED B4. If tanθ=3/4, find secθ.

Answer: 5/4

MED B5. Simplify sin²θ + cos²θ - 1.

Answer: 0

MED B6. Find sin45° and cos45°.

Answer: 1/√2

HARD B7. Prove 1+tan²θ=sec²θ.

Answer: Divide sin²θ+cos²θ=1 by cos²θ.

HARD B8. If sinθ = 12/13 and θ is acute, find all six trigonometric ratios.

Answer: cos=5/13, tan=12/5, cot=5/12, sec=13/5, cosec=13/12

HARD B9. Prove that (1+sin²θ)/cos²θ = 1+2tan²θ.

Answer: Expand and use identities

Quick Formula + Error Revision

  • sin²θ + cos²θ = 1
  • 1 + tan²θ = sec²θ
  • 1 + cot²θ = cosec²θ

Common mistakes to avoid

  • Mixing opposite/adjacent sides.
  • Wrong reciprocal pairs (sec↔cos, cosec↔sin, cot↔tan).
